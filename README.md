A Launchbar action to search gifs on [GIPHY](https://giphy.com)

---

![https://raw.githubusercontent.com/django-wong/launchbar-giphy/master/demo.gif](https://raw.githubusercontent.com/django-wong/launchbar-giphy/master/demo.gif)

#### Install

- Download zip file and open *GIPHY.lbaction*


#### Requirement

- You need a GIPHY API Key, get [here](https://developers.giphy.com)

#### Usage

- <kbd>GIPHY</kbd> > <kbd>Space</kbd> > Paste you API Key
- <kbd>GIPHY</kbd> > <kbd>Space</kbd> > Search keyword[:PAGE] > <kbd>Enter</kbd>
- <kbd>GIPHY</kbd> > <kbd>Right</kbd> to show trending gifs
- <kbd>GIPHY</kbd> > <kbd>Command + Right</kbd> > Remove API Key or cache files
- Instant send Giphy URL to GIPHY action > <kbd>Enter</kbd>
